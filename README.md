## Project Overview

This repository contains a Unity project using C#, designed for game development or interactive media.

### Software Purpose

- **Development Process:** This project uses an **Agile development process**. Agile was chosen over the waterfall model due to its flexibility and suitability for iterative design, rapid feedback, and accommodating evolving requirements in game/interactive media development.
- **Target Market:** Hobbyist gamers, independent developers, and educational environments seeking interactive Unity-based digital experiences.

## Software Development Plan

## Development & Runtime Environment

- **Programming Language:** C#
- **Engine:** Unity (recommended version: see Unity Hub/ProjectSettings)
- **Minimum Hardware:** Dual-core CPU, 4GB RAM, DX10 graphics card
- **Software Requirements:** Unity Hub, Visual Studio with Managed Game workload
- **Required Packages:** See `Packages/manifest.json` for Unity package dependencies.

## Getting Started

1. Clone this repository.
2. Open the project in Unity Hub (select the root folder).
3. If prompted, allow Unity to install/resolve packages automatically.
4. Open scripts with Visual Studio for editing and debugging.
